# Godot Tuneur

A Godot plugin to help me create rooms matching the style of Tunic.

Depends on [Godot GoBlend](https://github.com/Dams4K/GodotGoBlend)

See also [Blender Tuneur](https://github.com/Dams4K/BlenderTuneur)

## How to use

Add the file `res://addons/godot_tuneur/room_manager/room_manager.tscn` as an autoload.

Use the file `res://addons/godot_tuneur/importer/tuneur_scene_post_import.gd` as the post import file.

> [!CAUTION]
> This is not intended to be used by people other than me. Not because I'm selfish, it's just that I've made this tool to suit my needs.
